Portfólio



  • Cofrinho de Moedas (Java):

    Este projeto consiste em um pequeno sistema Java que simula um "Cofrinho de Moedas". Ele permite aos usuários adicionar, retirar e verificar o saldo de moedas armazenadas no cofrinho. Implementado como parte de um exercício de Programação Orientada a Objetos (POO), o programa utiliza conceitos como encapsulamento, herança e polimorfismo para criar uma interface interativa de gerenciamento de moedas.

  • Aplicativo de Vendas para Empresa X (Python):

    Este projeto representa a criação de um aplicativo de vendas desenvolvido em Python para a Empresa X, especializada em vendas por atacado. O aplicativo implementa uma estratégia de descontos escalonados com base na quantidade de produtos comprados. Através da captura do valor unitário do produto e da quantidade desejada pelo usuário, o programa calcula o valor total sem desconto e, em seguida, aplica descontos progressivos conforme a tabela fornecida pela Empresa X. Utilizando estruturas condicionais if, elif e else, o aplicativo demonstra a aplicação prática de lógica de programação para cálculo de preços e descontos em um contexto de vendas por atacado.

  • Sistema de Vendas de Pizzaria (Python):

    Neste projeto em Python, desenvolvi um sistema de vendas para uma pizzaria. O programa permite aos clientes escolherem o tamanho da pizza, selecionar diferentes sabores e adicionar acompanhamentos. Utilizando estruturas condicionais, loops e funções, o sistema calcula o valor total do pedido e fornece uma experiência interativa para realizar pedidos personalizados de pizzas.

  • Sistema de Pedidos de Feijoada (Python):

    Este projeto é um sistema de software desenvolvido em Python para um restaurante que serve feijoada. O programa oferece uma solução completa para a realização e cálculo de pedidos de feijoada, considerando diferentes opções de volume, tipos de feijoada e acompanhamentos. Implementando conceitos de programação estruturada, o sistema utiliza funções para calcular os valores com base nas opções escolhidas pelos clientes, garantindo precisão e eficiência na geração do valor total do pedido. Através de um menu interativo e da interação com o usuário, o sistema oferece uma experiência intuitiva de criação e cálculo de pedidos de feijoada personalizados, demonstrando habilidades na aplicação de lógica de programação para resolver desafios do mundo real.

  • Controle de Estoque para Mercearia (Python):

    Esse projeto consiste em um aplicativo de controle de estoque desenvolvido em Python. O aplicativo oferece funcionalidades como cadastrar produtos, consultar produtos por código ou fabricante, remover produtos e visualizar todos os produtos cadastrados. Com o uso de listas e dicionários, o programa demonstra a implementação de um sistema de gerenciamento de estoque simples, porém eficaz.
